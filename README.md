# LetsQuiz

My first project LetsQuiz. Simply app to game in Quiz. User can play, checks list of the top 10 the best scores, adds new questions and shows list of questions with possible answers.

# How to load?

  1. Download ZIP
  2. Open file main.py
  3. Compile the file 
  
# Used technologies

  1. Python
  2. Pytest
  3. Json
